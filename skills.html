MY SKILLS

  • HTML

    The foundation of web development, used to structure web content with elements like headings, paragraphs, and links.

  • CSS

    A styling language used to design and layout web pages, adding colors, fonts, spacing, and responsiveness.

  • JavaScript

    A programming language that adds interactivity and dynamic features to websites, such as forms, animations, and APIs.

  • React

    A JavaScript library for building user interfaces, focusing on reusable components and efficient rendering.

  • Node.js

    A runtime environment for executing JavaScript on the server side, enabling the creation of scalable backend systems.
